Exemplo: Método map() em Arrays

O método map() cria uma nova matriz executando uma função em cada elemento da matriz.

  • O map() não modifica o array original.
  • Não executa a função em elementos vazios.
  • Retorna uma nova array com os resultados.

Exemplo fixo:

Array original: [1, 2, 3, 4, 5]

Com map(): [2, 4, 6, 8, 10]
