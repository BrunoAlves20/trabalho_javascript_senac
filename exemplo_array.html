Exemplo 1: Método map() em Arrays

O método map() cria uma nova matriz executando uma função em cada elemento da matriz.

  • O map() não modifica o array original.
  • Não executa a função em elementos vazios.
  • Retorna uma nova array com os resultados.

Exemplo fixo:

Array original: [1, 2, 3, 4, 5]

Com map(): [2, 4, 6, 8, 10]

Exemplo 2: Método filter() em Arrays

O método filter() cria uma nova array contendo apenas os elementos que passam em um teste.

  • Ele não altera o array original.
  • Retorna uma nova array com os elementos que retornaram true na função de teste.
  • Quando usar filter()?
    Quando você quer remover elementos com base em uma regra.
    Quando quer criar uma nova lista com base em algum critério.
    Quando quer deixar seu código mais funcional e legível, ao invés de usar for com if.

Exemplo:

Array original: [1, 2, 3, 4, 5, 6]

Com filter(n => n % 2 === 0): [2, 4, 6]
